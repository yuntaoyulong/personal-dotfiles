# Personal Workflow Guides

这是我的长期工作流总入口，用于提醒我自己，也提醒 Codex。

## Core Stack
- `Hyprland`：系统级效率与窗口流
- `Neovim`：写作、代码、导出主编辑器
- `pptxws + Marp`：演示文档生产线
- `zsh + direnv + just`：终端与项目执行入口

## How To Use
- 每次开工先读 `principles.md`
- 做演示文档时看 `pptxws.md`
- 配置编辑器时看 `neovim.md`
- 调整 shell 时看 `zshrc.md`
- 查新装工具时看 `productivity-tools.md`
- IELTS 连续学习时看 `ielts.md`
- 调整桌面/快捷键时看 `hyprland.md`
- 交付前执行 `checklist.md`

## For Codex
- Codex 改动前先遵守 `principles.md`
- 优先维护现有结构，不随意重构
- 新增能力时，优先补充到对应 guide 文件
