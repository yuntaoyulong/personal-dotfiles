# Delivery Checklist

## Before Starting
- [ ] 明确当前任务属于 `pptxws` / `nvim` / `hyprland` 哪一类
- [ ] 选择对应 guide 文件作为准则

## During Changes
- [ ] 配置与脚本都可读、可维护
- [ ] 不引入重复路径和重复逻辑
- [ ] 能命令化就不手工化

## After Changes
- [ ] 命令真实执行通过（不是只看配置文件）
- [ ] 输出产物真实存在（如 `exports/*.pptx`）
- [ ] 快捷键真实可触发（Hypr 会话内验证）
- [ ] 必要时补一条 guide 经验，避免再次踩坑
